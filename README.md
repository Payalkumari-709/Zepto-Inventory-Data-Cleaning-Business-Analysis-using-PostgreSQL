

# Zepto Inventory Data Cleaning & Exploratory Data Analysis using PostgreSQL

## 📌 Project Overview

This project focuses on analyzing Zepto's retail inventory dataset using PostgreSQL. The objective is to perform data exploration, data cleaning, exploratory data analysis (EDA), business analysis, and advanced SQL analysis to derive meaningful business insights from the dataset.

The project demonstrates the complete SQL workflow followed by a Data Analyst, from understanding raw data to generating actionable business insights.

---

## 📂 Dataset Information

The dataset contains inventory information for products available on Zepto, including:

- Product Name
- Category
- Maximum Retail Price (MRP)
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Product Weight
- Stock Availability
- Quantity

## 💡 Key Business Insights

• Cooking essentials category has the largest inventory.

• Fruits and Vegetables offer the highest average discount.

• Munchies category have highest number of out of stock product.

• Some high-MRP products remain out of stock, indicating possible inventory issues.

• Several products provide excellent value due to high discounts and competitive price-per-gram.
